# flutter_uwb brand assets

Production-ready visual identity for `flutter_uwb`. Use as-is.

## Files

| Asset                              | Dimensions   | Use                                                |
| ---------------------------------- | ------------ | -------------------------------------------------- |
| `flutter_uwb_pulse.svg`            | 320×320 vbox | Primary mark — concentric rings + tracked-point dot, transparent edges over the brand dark plate. |
| `flutter_uwb_pulse_1024.png`       | 1024×1024    | Rasterised mark for app launcher icons (`flutter_launcher_icons`). |
| `flutter_uwb_banner.png`           | 2400×600     | Horizontal promo banner. First image in the README → pub.dev social preview. |
| `flutter_uwb_screenshot.png`       | 1200×1600    | Vertical phone-mockup promo for the README hero. |
| `badges/flutter_uwb_badge_shields.svg` | 220×32   | Two-tone shields-style badge (`flutter_uwb · UWB`). |
| `badges/flutter_uwb_badge_solid.svg`   | 180×32   | Single dark plate badge — for "Powered by" footers. |
| `badges/flutter_uwb_badge_pill.svg`    | 200×32   | Cyan-bordered pill badge — alternate compact style. |

## Brand tokens

| Token         | Value     |
| ------------- | --------- |
| Background    | `#0A0E21` |
| Primary       | `#00E5FF` |

| Text          | `#E6ECFF` |
| Muted         | `#6B7392` |
| UI font       | Inter     |
| Code font     | JetBrains Mono |

## Usage rules

- Don't recolor. Don't recompose. Don't add gradients or drop shadows.
- Min clear-space around the mark = logo height × 0.25.
- The mark sits on the brand dark plate (`#0A0E21`) — don't place it on a light background.
